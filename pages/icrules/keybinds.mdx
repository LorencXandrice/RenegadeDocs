# §2 ⌨️ Useful Muscles ⌨️

<Callout type="info">
If you need to reset your keybinds, follow the steps below.
1. Completely close FiveM
2. *Press* `Windows` + `E` and *search* **`AppData\Roaming\CitizenFX`** in your `📁 Windows Explorer`
3. *Open* the `fivem.cfg` file in Notepad or a text editor
4. *Delete* every line that starts with `rbind`
5. *Save* the file and *restart* <font color="#FF8C3E"> **FiveM** </font>
</Callout>
