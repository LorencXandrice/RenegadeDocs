# §2 ⌨️ Useful Muscles ⌨️

Listed below are the most commonly used muscles. Feel free to change these any time by  
*pressing* `ESC` → `Settings` → `Key Bindings` → `FiveM`

| **Action**                                 | **Keybind**                 |
| :----------------------------------------- | :--------------------------:|
| 🔈🔉🔊 Change Voice Volume               | `F11`                       |
| 🎙️ Push to Talk                           | `N`                         |
| 📟 Talk over Radio                        | `Left Alt`                  |
| ⬆️ Increase Radio Channel                 | `F3`                        |
| ⬇️ Decrease Radio Channel                 | `F4`                        |   
| 💭 Text Chat                              | `T`                         |
| 📱 Open Phone                             | `M`                         |
| 🗺️ Open Map                               | `P` or `ESC`                |
| 👁️ 3rd Eye                                | `Left Alt`                  |
| ⭕ Open Menu Wheel                        | `F1`                        |
| 🎒 Open Player Inventory                  | `Tab`                       |
| 🚗 Open Secondary Inventory (e.g., trunk) | `~`                         |
| 👉 Point Finger                           | `B`                         |
| 🫷🫸 Hands Up                             | `X`                         |
| 🙇 Ragdoll                                | `U`                         |
| 🚶‍➡️🙇Tackle                                | `Shift` + `E` while moving  |
| 🛍️ Make a Sale                            | `H`                         |
| 🕺💃 Open Emote Menu                      | `F4`                        |
| ❎ Cancel Emote / Progress Bar            | `X`                         |
| 🚗 Seatbelt                               | `B`                         |
| 🚗 Toggle Engine                          | `G`                         |
| 🚗 Toggle Headlights                      | `H`                         |
| 🚗 Lock Car                               | `L`                         |
| 🧑‍💻 Show Player IDs                        | `F10`                       |
