# 🖮 Keybinds

Listed below are the most commonly used keybinds. Feel free to change these any time by\
*pressing* `ESC` → `Settings` → `Key Bindings` → `FiveM`

| **Action**                                 | **Keybind**                |
| :----------------------------------------- | :------------------------: |
| 🎙️ Push to Talk                           | `N`                         |
| 🔈🔉🔊 Change Voice Volume               | `                           |    
| 💭 Text Chat                              | `T`                         |
| 📱 Open Phone                             | `M`                         |
| 👁️ 3rd Eye                                | `Left Alt`                  |
| ⭕ Open Menu Wheel                        | `F1`                        |
| 🕺💃 Open Emote Menu                      | `F4`                       |
| 🗺️ Open Map                               | `P` or `ESC`                |
| 🎒 Open Player Inventory                  | `Tab`                       |
| 🚗 Open Secondary Inventory (e.g., trunk) | `~`                         |
| 🚗 Seatbelt                               | `B`                         |
| 🚗 Toggle Engine                          | `G`                         |
| 🚗 Toggle Headlights                      | `H`                         |
| 🚗 Lock Car                               | `L`                         |
| 👉 Point Finger                           | `B`                         |
| 🫷🫸 Hands Up / Cancel Emote              | `X`                        |
| 🙇 Ragdoll                                | `U`                         |
| 🚶‍➡️🙇Tackle                                | `Shift` + `E` while moving |
| 🧑‍💻 Show Player IDs                        | `F10`                       |
