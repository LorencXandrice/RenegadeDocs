Fivemanage Custom Uploader for ShareX
This tutorial will guide you through integrating ShareX with our service. By following these steps, you'll be able to upload images directly to our service using ShareX. I'll also provide a tip on how to automate the process to upload images right after capture and copy their URLs to the clipboard for easy sharing.
Prerequisites
An image or media API key generated from your Fivemanage dashboard.
Configuration Steps
Prepare the Custom Uploader
Copy the JSON configuration provided below. Replace YOUR_API_KEY_HERE with the API key generated through your Fivemanage dashboard.

```json
{
  "Version": "15.0.0",
  "Name": "Fivemanage",
  "DestinationType": "ImageUploader",
  "RequestMethod": "POST",
  "RequestURL": "https://api.fivemanage.com/api/image",
  "Headers": {
    "Authorization": "YOUR_API_KEY_HERE"
  },
  "Body": "MultipartFormData",
  "FileFormName": "image",
  "URL": "{json:url}",
}
```
Import the Custom Uploader to ShareX
Right-click the ShareX icon in your system tray.
Navigate to Destinations > Custom Uploader Settings.
Click on Import > From Clipboard to load the configuration into ShareX.
Set our new Custom Uploader as the default for Image Uploads
Right-click the ShareX icon in your system tray again.
Navigate to Destinations > Image uploader.
Select Custom image uploader from the list.
You're good to go! You can now upload images to Fivemanage through ShareX!!!

Upload Image After Capture and Copy URL to Clipboard
Upload Image After Capture
Right-click the ShareX icon in your system tray.
Select After capture tasks.
Make sure Upload image to host is checked.
Copy URL to Clipboard
Right-click the ShareX icon again.
Select After upload tasks.
Make sure Copy URL to clipboard is checked
Now after capturing an image it is automatically uploaded to the Fivemanage servers and the URL is copied to your clipboard.
